В данном справочнике хранится информация о структуре объектов проверяемой конфигурации в разрезе версий. За каждый объект версии может быть назначен ответственный: найденные недочеты по этому объекту автоматически будут распределены на него. Для новой версии ответственные назначаются автоматически после сбора данных по назначенным ответственным за последнюю проверяемую версию.

Ответственные могут быть назначены по информации из файла Excel, в котором в первой колонке указан тип объекта, во второй - имя объекта, а в третьей – ответственный. При проверке объектов, в случае не назначенного ответственного за объект, будет взят первый заполненный ответственный по мере следования "родителей" объекта.

В случае отсутствия ответственных и у них, в найденных недочетах будет назначен ответственный за конфигурацию.
