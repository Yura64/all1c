Для хранения списка ответственных предназначен справочник «Ответственные». Если объекту в структуре конфигурации назначить ответственного, то для всех найденных недочетов в этом объекте будет установлен этот ответственный.

Для новой версии ответственные назначаются автоматически после сбора данных по назначенным ответственным за последнюю проверяемую версию.

Ответственные могут быть назначены по информации из файла в котором в первой колонке указан тип объекта, во второй - имя объекта, а в третьей – ответственный.
