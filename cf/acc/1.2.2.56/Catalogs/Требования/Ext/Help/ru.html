После завершения корректировки состава и содержания правил корректируются программы проверки. Это выполняется в форме списка справочника «Программы проверки», которая вызывается из пункта «Программы проверки» подменю «Настройка» главного меню. Программы проверки выводятся при настройке проверки в соответствии с иерархией, заданной в данном справочнике.

Корректировка программы проверки выполняется в форме элемента. Для программы проверки необходимо указать лаконичное и понятное название, составить описание программы проверки: её назначение и особенности и после этого определить состав. Для определения состава необходимо выбрать те правила проверки в поле «Все правила», которые будут составлять данную программу.

При выборе правило появляется в поле «Состав». Для исключения правила из состава программы проверки необходимо выбрать его в поле «Состав».

Программа (требование) содержит поле "Способ проверки" (или "Проверка"), которое заполняется одним из следующих значений:

- Автоматически, если все правила, входящие в состав требования, автоматизированы;

- Полуавтоматически, если в составе требования есть хотя бы одно автоматизированное правило и хотя бы одно ручное;

- Вручную, если все правила в составе требования являются ручными;

- В разработке, если в данный момент происходит разработка автоматизированного правила из состава данного требования - временный статус.
