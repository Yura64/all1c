Обработка "Загрузка ответственных" предназначена для загрузки ответственных в другую конфигурацию.

Перенос ответственных может происходить как между конфигурациями в одной базе данных АПК, так и между различными базами АПК.

Сопоставление ответственных происходит по наименованию, объектов - по полному пути.

Перенос осуществляется через файл MS Excel формата ".xlsx" или ".xls".

Считывание данных при загрузке происходит из книги Excel с первого листа, начиная с левого верхнего угла.

 

Заполнение колонок:

Колонка A (1) - Ответственный

Колонка B (2) - Путь к объекту

Колонка C (3) - Тип объекта

 

В первой строке допустимы заголовки колонок. Если заголовки есть, то загрузка производится со второй строки.

Если при загрузке не найден ответственный, то этот объект не будет обработан, выйдет сообщение о том, что данная строка файла пропущена. Если у объекта метаданных уже есть ответственный, он сменится на ответственного из файла. Если производится смена ответственного у подсистемы, то он сменится у всех объектов метаданных, входящих в эту подсистему.

Если при загрузке не найден объект метаданных по указанному полному пути из файла, то произойдет повторный поиск объекта в ветке "Общие". Если объект не найден и там, то произойдет еще один поиск по наименованию. Если объект не найден, то он будет пропущен, и об этом будет сообщено пользователю.

Если на каком-то этапе поиска будет найдено несколько объектов, удовлетворяющих поиску, то к фильтру отбора будет подключен тип объекта. Если и после этого объект не удастся определить однозначно, то пользователю будет сообщено об этом, и строка файла будет пропущена.
