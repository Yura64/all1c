Обработка "Выгрузка ответственных" предназначена для выгрузки ответственных за объекты метаданных конфигурации.

Осуществляется через файл MS Excel формата ".xlsx" или ".xls".

Выгрузка производится в книгу Excel на первый лист, начиная с левого верхнего угла.

 

Заполнение колонок:

Колонка A (1) - Ответственный

Колонка B (2) - Путь к объекту

Колонка C (3) - Тип объекта

 

По умолчанию, формирование файла с ответственными производится с заголовками колонок.

Если при выгрузке ответственных указан существующий файл, он заменится. Если его предварительное удаление не удалось, то выйдет сообщение об ошибке, и выгрузка не произойдет.

В файл выгружаются только те ответственные, которые явно указаны для объекта метаданных в справочнике "Структура конфигурации". Унаследованные ответственные от других объектов не выгружаются.
